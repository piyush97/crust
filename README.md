# Crust

Crust is a Golang API for dealing with Auth and e-commerce transactions and checkouts.

## Installation


```bash
git clone https://github.com/piyush97/crust
cd crust
# run from golang tools
go run main.go 
# or use make
make
```

## Usage

```bash
# base url
/api 
# user api
/api/user
/api/user/register
/api/user/signin

```
